# Human-in-the-Loop AI for Flood Risk Prediction

## Overview

This project investigates how Human-in-the-Loop (HITL) techniques improve the reliability of AI-based flood risk prediction.

A Random Forest classifier was trained on a flood-risk dataset and evaluated through five research questions.

## Research Questions

- RQ1: AI-only prediction performance
- RQ2: Impact of human review
- RQ3: Critical prediction errors
- RQ4: Feature importance analysis
- RQ5: Accuracy versus review workload
